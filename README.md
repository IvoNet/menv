# menv

Environment setter for maven.

With this command you can set a folder (structure) to a specific maven profile and maven opts.


# Install

With [Homebrew](https://brew.sh)

```bash
brew tap ivonet/cli
brew install menv
```


# Usage

```bash
brew -h
```
